We can even view
$
\frac{1}{\sqrt{n}}
\sum_{i=1}^{n}S_i 
$
as an empirical process 
$
\mathbb{G}_n f
$
indexed over 

\begin{gather}
  f_\Phi(T,X,Y)
  =
  \frac{T}{\pi(X)}
 \left( 
   \Phi(Y) - \E [ \Phi(Y) | X ]
 \right)
 +
   \E [ \Phi(Y) | X ] 
   .
\end{gather}
If $\mathcal{F}=\left\{ f_\Phi \colon \Phi \in \ \text{some set}\right\}$
is $\P$-Donsker, the empirical process converges to a tight gaussian process.
Then the functional delta Method is applicable.
